AI Resume Analyzer

AI Resume Screener

A simple Full-Stack Web Application built using Python (Flask), HTML, and CSS that analyzes a resume and matches it with a job description based on skill extraction.

-- Features
Upload or paste Resume text
Paste Job Description
Extracts skills automatically
Calculates match percentage
Shows matching and missing skills
Clean and simple web UI

-- Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Logic: Basic NLP (Skill Matching)

-- Installation & Setup
Install dependencies: pip install flask
Run the project: python app.py
Open in browser: http://127.0.0.1:5000/

-- How It Works
User enters Resume text
User enters Job Description
Python extracts skills from both
System compares skills
Match percentage is calculated
Results are displayed on screen

-- Example Output
Match Score: 85%
Resume Skills: Python, Java, HTML, CSS
Job Skills: Python, Java, SQL, Flask
Missing Skills: SQL, Flask
